# ⏳ Loading Screen

> Uma tela de carregamento animada com spinner circular, ideal para indicar processos em execução.

---

## 📋 Sobre o Projeto

A **Loading Screen** é um componente visual que exibe um indicador de carregamento (spinner) animado sobre um fundo colorido, comumente usado enquanto uma página, aplicação ou requisição está processando.

---

## 🚀 Funcionalidades

- Tela de carregamento em tela cheia
- Spinner circular com animação de rotação
- Fundo em gradiente azul

---

## 🗂️ Estrutura do Projeto

```
LOADING SCREEN/
├── index.html     # Estrutura da página
└── style.css      # Estilização e animação do spinner
```

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura da página
- **CSS3** — Animação do spinner (`@keyframes`) e estilização do fundo

---

## ▶️ Como Executar

1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` diretamente no navegador

Ou use uma extensão como **Live Server** no VS Code para rodar em `http://127.0.0.1:5500`.

---

## 📐 Como Funciona

1. A tela é exibida em tela cheia com fundo azul
2. O spinner circular gira continuamente através de animação CSS
3. Pode ser usado como tela inicial enquanto outros recursos da página carregam

---

## 🎨 Personalização

É possível alterar facilmente a cor do fundo, o tamanho e a velocidade do spinner editando as variáveis no `style.css`.
